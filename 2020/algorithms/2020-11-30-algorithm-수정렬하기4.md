# 11931번. 수 정렬하기 4

### 문제 링크
- <https://www.acmicpc.net/problem/11931>{: target="\_blank"}

### 풀이 코드

```python
# 11931번. 수 정렬하기 4


import sys
input = sys.stdin.readline

n = int(input())
nList = []
for i in range(n):
    nList.append(int(input()))
nList.sort(reverse=True)
for num in nList:
    print(num)
```
