# Variable LifeCycle

### member variable

- class 내에 정의되는 variable
- 메모리 영역 중 heap에 생성된다.
- static(class) variable : class가 메모리에 로딩시에 메모리 할당이 이루어지며, 자동초기화 된다. class 제거 시 제거된다.
- instance variable : 객체 생성 시 자동으로 초기화되며, 객체 제거시 제거된다.
